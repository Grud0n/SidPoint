Javascript

- Programmiersprache für dynamisches HTML im Webbrowser
- 1995 von Netscape entwickelt
Verwendung von Javascript in unserem Projekt:
- Steuerung der Navigation durch die Folien der Präsentation sowie das Anzeigen von Inhalten
Die Funktion showMessage(section) wird definiert, um die Nachrichten zu steuern, die angezeigt werden, wenn auf die Schaltflächen geklickt wird, um zwischen den Folien zu navigieren.
Die Variable currentSection wird initialisiert, um die aktuell sichtbare Folie zu verfolgen. Standardmäßig ist die erste Folie (ID 1) sichtbar.
Innerhalb der showMessage-Funktion werden die aktuellen Abschnitte ausgeblendet und die entsprechenden Schaltflächen aktualisiert, um den Status der Navigation anzuzeigen.
Wenn die Funktion mit "Left" oder "Right" aufgerufen wird, wird die nächste bzw. vorherige Folie entsprechend der aktuellen currentSection ausgewählt.
Wenn die Funktion mit einer Nummer aufgerufen wird, wird direkt zur entsprechenden Folie gesprungen.
Nachdem die neue aktive Folie ausgewählt wurde, wird sie sichtbar gemacht und die zugehörige Schaltfläche wird als "ausgewählt" markiert.
Zusätzlich wird überprüft, ob die aktuellen Folien die ersten oder letzten sind, um die Pfeilschaltflächen entsprechend auszublenden oder anzuzeigen.
Schließlich wird beim Laden der Seite die Funktion showMessage(1) aufgerufen, um die erste Folie anzuzeigen.

HTML

- textbasierte Auszeichnungssprache zur Strukturierung elektronischer Dokumente
HTML = Hypertext Markup Language
- 1992 von World Wide Web Consortium veröffentlicht
Der HTML-Code definiert die Struktur und den Inhalt einer Webseite.
head-Bereich werden Metadaten wie Zeichensatz und Verweise auf externe Dateien festgelegt.
Im body-Bereich befinden sich die eigentlichen Inhalte der Webseite, einschließlich Header, Hauptinhalt und Footer.
Der Header enthält die Top-Leiste mit Schaltflächen und dem Titel der Präsentation.
Der Hauptinhalt besteht aus Abschnitten, die die einzelnen Folien der Präsentation darstellen.
Jede Folie hat eine eindeutige ID und besteht aus einem Titelfeld und einem Textfeld.
Der Footer enthält das Datum und den Eigentümer der Präsentation.
